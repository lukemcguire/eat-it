    2. POST / (create empty list):
       - Accept ShoppingListCreate
       - Create ShoppingList with name
       - Return 201 with ShoppingListPublic

    3. GET / (list all lists):
       - Return ShoppingListsPublic with all lists (no items nested)
       - Order by created_at desc

    4. GET /{list_id} (get single list with items):

    5. PATCH /{list_id} (update list name):

       - Return updated ShoppingListPublic

    6. DELETE /{list_id} (delete list):
       - Delete list and all items (cascade)
       - Return 204

    7. POST /{list_id}/items (add item):
       - Accept ShoppingListItemCreate
       - Auto-categorize to section if section_id not provided
       - Create ShoppingListItem
       - Return 201 with ShoppingListItemPublic

    8. PATCH /{list_id}/items/{item_id} (update item):
       - Accept ShoppingListItemUpdate
       - Update only provided fields
       - Return updated ShoppingListItemPublic

    9. DELETE /{list_id}/items/{item_id} (remove item):
       - Delete item
       - Return 204

    10. POST /generate (generate from recipes):
        - Accept ShoppingListGenerate with name and recipe_ids
        - For each recipe_id, get recipe and collect all ingredient raw strings
        - Call combine_ingredients() with all raw strings
        - Create ShoppingList with name
        - For each CombinedIngredient:
          - Categorize to section
          - Create ShoppingListItem with combined data
        - Return 201 with ShoppingListPublic (including items)

    For generate endpoint:
    ```python
    @router.post("/generate", response_model=ShoppingListPublic, status_code=201)
    def generate_shopping_list(
        data: ShoppingListGenerate,
        session: Session = Depends(get_session)
    ):
        # Collect all ingredient raw strings from recipes
        all_ingredients = []
        for recipe_id in data.recipe_ids:
            recipe = session.get(Recipe, recipe_id)
            if not recipe:
                continue  # Skip invalid recipe IDs
            groups = session.exec(
                select(IngredientGroup).where(IngredientGroup.recipe_id == recipe_id)
            ).all()
            for group in groups:
                ingredients = session.exec(
                    select(Ingredient).where(Ingredient.group_id == group.id)
                ).all()
                all_ingredients.extend([ing.raw for ing in ingredients])

        # Combine ingredients
        combined = combine_ingredients(all_ingredients)

        # Create list
        db_list = ShoppingList(name=data.name)
        session.add(db_list)
        session.commit()
        session.refresh(db_list)

        # Create items with auto-categorization
        for item in combined:
            section_name = categorize_ingredient(item.name)
            section_id = get_or_create_section_id(session, section_name)
            db_item = ShoppingListItem(
                shopping_list_id=db_list.id,
                name=item.name,
                quantity=item.quantity,
                unit=item.unit,
                section_id=section_id
            )
            session.add(db_item)

        session.commit()
        session.refresh(db_list)
        return db_list
    ```
