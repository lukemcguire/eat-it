       - CombinedIngredient dataclass: name, quantity, unit, raw_strings (list of source strings)
       - combine_ingredients(ingredient_strings: list[str]) -> list[CombinedIngredient]
         - Uses ingredient_parser.parse_ingredient with string_units=True
         - Normalizes names: lowercase, strip whitespace, ignore preparation notes
         - Groups by normalized_name + unit
         - Sums quantities for same group
         - DOES NOT combine similar names (onion != red onion per CONTEXT.md)
         - Returns list sorted by name

    Key implementation details from RESEARCH.md:
    - Use parse_ingredient(raw, string_units=True) for unit handling
    - result.name[0].text for ingredient name
    - result.amount[0].quantity and result.amount[0].unit for amounts
    - Handle ingredients without amounts (e.g., "salt to taste")

    Example:
    ```python
    from ingredient_parser import parse_ingredient
    from dataclasses import dataclass
    from collections import defaultdict

    @dataclass
    class CombinedIngredient:
        name: str
        quantity: float | None
        unit: str | None
        raw_strings: list[str]

    def combine_ingredients(ingredient_strings: list[str]) -> list[CombinedIngredient]:
        combined = defaultdict(lambda: {"quantity": 0.0, "unit": None, "raw": []})
        for raw in ingredient_strings:
            result = parse_ingredient(raw, string_units=True)
            name = result.name[0].text.lower().strip() if result.name else raw.lower().strip()
            if result.amount:
                qty = float(result.amount[0].quantity) if result.amount[0].quantity else 0
                unit = result.amount[0].unit if isinstance(result.amount[0].unit, str) else str(result.amount[0].unit) if result.amount[0].unit else None
                key = f"{name}|{unit}"
                combined[key]["quantity"] += qty
                combined[key]["unit"] = unit
            else:
                key = f"{name}|"
                combined[key]["unit"] = None
            combined[key]["name"] = name
            combined[key]["raw"].append(raw)
        return [CombinedIngredient(name=v["name"], quantity=v["quantity"] if v["quantity"] else None, unit=v["unit"], raw_strings=v["raw"]) for v in combined.values()]
    ```
