<task type="auto">
  <name>Task 2: Create embedding service module</name>
  <files>src/eat_it/services/embedding.py</files>
  <action>
    Create a new service module for embedding operations at src/eat_it/services/embedding.py.

    The module should provide:

    1. `serialize_f32(vector: list[float]) -> bytes`
       - Serialize a list of floats into compact binary format for sqlite-vec
       - Use struct.pack("%sf" % len(vector), *vector)
       - This is the format sqlite-vec expects for BLOB storage

    2. `generate_embedding(text: str, model: SentenceTransformer) -> list[float]`
       - Generate embedding for text using the provided model
       - Call model.encode(text, convert_to_numpy=True) and convert to list
       - Handle None model gracefully (raise RuntimeError)

    3. `recipe_to_text(recipe: Recipe) -> str`
       - Convert recipe to text for embedding
       - Combine: title + description (if exists) + instructions
       - Exclude ingredients (per RESEARCH.md - focus on dish identity)
       - Use space as separator

    Import struct at the top. Add proper type hints and docstrings.

    Reference RESEARCH.md Pattern 2 for the implementation patterns.
  </action>
  <verify>

  </verify>
