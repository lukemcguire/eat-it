  <action>
    Create src/eat_it/models/recipe.py with three models:

    **Ingredient model (table=True):**
    - id: Optional[int] primary key
    - group_id: int (foreign key to ingredient_groups.id)
    - quantity: Optional[float] (can be null)
    - unit: Optional[str] (can be null for "salt to taste")
    - name: str
    - preparation: Optional[str] (e.g., "diced", "minced")
    - raw: str (original unparsed string)
    - created_at, updated_at: datetime

    **IngredientGroup model (table=True):**
    - id: Optional[int] primary key
    - recipe_id: int (foreign key to recipes.id)
    - name: Optional[str] (e.g., "For the sauce:")
    - created_at, updated_at: datetime

    **Recipe model (table=True):**
    - id: Optional[int] primary key
    - title: str
    - description: Optional[str]
    - instructions: str
    - prep_time: Optional[int] (minutes)
    - cook_time: Optional[int] (minutes)
    - servings: Optional[int]
    - source_url: Optional[str]
    - image_url: Optional[str]
    - tags: List[str] with sa_column=Column(JSON)
    - metadata_: dict with sa_column=Column("metadata", JSON) and
      default_factory=lambda: {"version": 1, "data": {}}
    - created_at, updated_at: datetime

    IMPORTANT: Use metadata_ as field name (SQLAlchemy reserves metadata).
    Add indexes on source_url and created_at.

    Create src/eat_it/models/__init__.py exporting all models.
  </action>
